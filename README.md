- Quản lý email
- Quản lý khách hàng
- Quản lý bán hàng
- Quản lý nhà cung cấp
- Quản lý báo giá
- Quản lý vật tư

- Quản lý kho
- Quản lý giá bình quân gia quyền
- Tính lương
- Chấm công
- Lập kế hoạch
